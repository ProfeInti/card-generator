# Legacy Local Backend

Esta carpeta conserva un backend local antiguo basado en Express + SQLite.
El flujo activo actual del proyecto usa React + Supabase y no depende de estos archivos para funcionar.

Notas:
- `card-generator.db`, `card-generator.db-shm` y `card-generator.db-wal` son artefactos locales y no deben versionarse.
- Si en el futuro se elimina esta carpeta, conviene hacerlo en una fase aparte y verificada.
